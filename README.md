# SAM x Nano4M — Quarto website

This repository contains a Quarto website for the project SAM x Nano4M in COM-304 prepared for GitHub Pages.


# To update the website 

1. install quarto
2. clone this repo
3. make the modifications
4. run following command  ```$ quarto render ``` 
5. add the output files (qmd + json + html), commit and push
6. wait a small delay, the page will re-actualize in a short amount of time
