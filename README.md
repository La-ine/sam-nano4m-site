# SAM x Nano4M — Quarto website

This repository contains a Quarto website prepared for GitHub Pages.

## Publish

1. Install Quarto: https://quarto.org
2. From this folder, run:
   ```bash
   quarto render
   ```
3. Push the repository to GitHub.
4. In GitHub Pages settings, deploy from the `docs/` folder on the main branch.

## Customize

- Put your real images in `images/`
- Replace the GitHub link in `_quarto.yml`
- Add your report PDF if desired
